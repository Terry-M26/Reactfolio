# Terry Mardaymootoo — Portfolio

Personal developer portfolio showcasing my projects, skills, and experience as an aspiring Software Developer.

## About

This is my personal portfolio website built with **React**, **Vite**, and **TailwindCSS**. It features interactive project detail pages with image carousels, animated UI elements, and dual light/dark theme support.

## Tech Stack

- **React** — UI framework
- **Vite** — Build tool
- **TailwindCSS** — Styling
- **React Router** — Client-side routing
- **React Icons** — Icon library

## Features

- Responsive design with light and dark themes
- Animated entrance effects and interactive UI elements
- Project detail pages with image carousels, feature lists, and tech breakdowns
- Skills section with labeled icons
- Contact section with email and social links
